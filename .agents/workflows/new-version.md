description: The steps to apply when the user asks for a new version
---

When you work on a new version:
1. Look at the older version(s) to see what worked and what went wrong
2. Look at all the options available in the 3 configurations files and find the ones that would help the most with the main issue(s)
3. Find the best values for the options that can impact the issues positively while avoiding negative side effects
4. Create a new folder for this version with only a process.md and changelog.md with the updated values
