Your priorities are:
1. Maintain >99% success print rate
2. Make sure the supports are easy to remove, that they snap easily and that they don't fuse with the print
2. Improve print quality and details
3. Reduce print time and quatity of plastic used (cost)

A print is based on 3 configurations:
- printer configuration
- filament configuration
- process configuration

When you add a feedback on a version:
1. Verify the photos available in /versionNumber/feedback
2. If there are no photos available, ask the user to add some
3. Use the feedbacks from the user and the photos to understand the issue, then copy the default feedback.md file and fill it

When you work on a new version:
1. Look at the older version(s) to see what worked and what went wrong
2. Look at all the options available in the 3 configurations files and find the ones that would help the most with the main issue(s)
3. Find the best values for the options that can impact the issues positively while avoiding negative side effects
4. Create a new folder for this version with only a process.md and changelog.md with the updated values
